# archive/vite-shell

Snapshot of the original Vite + React SPA prototype, archived on
`feat/v0.1-rebuild` (2026-04-20) as part of the canonical foundation
rebuild per [22] Phase — Canonical Beta Foundation.

Preserved as reference input for:

- Tailwind tokens and design language
- Component shape and layout patterns
- Domain type sketches (see `src/types/index.ts`)
- Visual conventions for booking lifecycle state and conflict state
- `src/utils/time.ts` nightlife-day helpers

Not runnable as-is. `node_modules/` is not included and `package.json` is
kept as reference only. The live runtime at the repo root is now Next.js
App Router, per the locked stack in `docs/source-of-truth.md`.

For the canonical rebuild target, see:

- `docs/phases/22-canonical-beta-foundation.md`
- `docs/phases/v0.1-rebuild-execution-plan.md`
